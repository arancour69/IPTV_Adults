## 💻 Installation Guides

### VLC Media Player (All Platforms)

**Desktop (Windows, Mac, Linux):**
1. Open VLC Media Player
2. Go to **Media** → **Open Network Stream**
3. Paste your M3U8 URL
4. Click **Play**

**Android:**
1. Download VLC from Google Play Store
2. Launch the app
3. Tap the menu icon
4. Select **Open Network Stream**
5. Paste M3U8 URL and play

### Kodi Media Center

**Installation:**
1. Open Kodi on your device
2. Go to **Settings** → **Add-ons**
3. Select **Install from repository**
4. Choose an IPTV add-on (e.g., "PVR IPTV Simple Client")
5. Install and configure with your M3U8 URL

**Configuration:**
1. Go to **Live TV** settings
2. Enable **Live TV**
3. Add your M3U8 playlist URL
4. Restart Kodi

### Tivimate (Android)

**Installation:**
1. Install Tivimate from APK or Play Store
2. Open the app
3. Tap **+** to add new playlist
4. Select **M3U Playlist**
5. Enter your M3U8 URL
6. Name and save

**Configuration:**
- Set preferred video quality
- Configure EPG (Electronic Program Guide)
- Customize channel organization

### IPTV Player

**Installation:**
1. Download IPTV Player for your platform
2. Open the application
3. Go to **Settings** → **Add Playlist**
4. Choose **M3U Format**
5. Enter your URL or import file
6. Apply and browse channels

## 🔧 Troubleshooting

### Streams Not Loading
1. ✅ Check internet connection (test speed at speedtest.net)
2. ✅ Restart your player application
3. ✅ Try a different playlist file
4. ✅ Clear player cache/temporary files
5. ✅ Update player to latest version
6. ✅ Check firewall/VPN settings

### Buffering Issues
1. Check bandwidth usage (close other apps)
2. Try switching to lower quality
3. Restart your router/modem
4. Consider wired Ethernet connection
5. Try a different playlist file

### Player Crashes
1. Update to latest player version
2. Restart your device
3. Try a different player
4. Check available disk space
5. Verify system compatibility

### Audio/Video Sync Issues
1. Pause and resume playback
2. Try different video quality
3. Update media drivers
4. Restart player application

## 📋 File Format

All playlist files use standard M3U8 format:
```
#EXTM3U
#EXT-X-VERSION:3
#EXT-X-TARGETDURATION:10
#EXT-X-MEDIA-SEQUENCE:0

#EXTINF:-1, Channel Name
http://stream.url/channel.m3u8

#EXTINF:-1, Another Channel
http://another.stream/channel.m3u8
```

**Learn more**: [M3U8 Format Documentation](https://en.wikipedia.org/wiki/M3U)
