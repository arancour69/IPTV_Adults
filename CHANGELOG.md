## Known Issues & Workarounds ⚠️

### Stream Loading Failures
- **Issue**: Some streams fail to load intermittently
- **Workaround**: Retry after 30 seconds or try different playlist
- **Status**: Under investigation

### Buffering in HD Quality
- **Issue**: Occasional buffering on slower connections
- **Workaround**: Switch to lower quality or use wired connection
- **Status**: Network dependent

### Android Compatibility
- **Issue**: Some older Android versions have compatibility issues
- **Workaround**: Update player to latest version
- **Status**: Player vendor support needed

### Playlist Refresh Delays
- **Issue**: Updated streams may take time to propagate
- **Workaround**: Manually refresh player playlist
- **Status**: Normal GitHub CDN behavior

## Migration Guides

### Upgrading from 1.0 to 2.0
1. Download new playlist files (AdultsHD_1 through AdultsHD_5)
2. Replace old files in your player
3. Restart player application
4. Enjoy improved stability!

### Upgrading from 2.0 to 2.1
1. Read new [README.md](README.md) for updated guides
2. Check [SUPPORT.md](SUPPORT.md) for troubleshooting
3. Review [CONTRIBUTING.md](CONTRIBUTING.md) for how to help
4. No player changes required

## Release Process

### How Versions Are Released
1. Changes accumulate on the main branch
2. Version bump when significant updates ready
3. CHANGELOG updated with all changes
4. Tagged release on GitHub
5. Announcement in discussions

### Version Numbering
- **MAJOR**: Breaking changes or complete rewrites
- **MINOR**: New features or improvements
- **PATCH**: Bug fixes and small updates
